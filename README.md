# TaoDialog
Dialog component based on tao framework.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'tao_dialog'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install tao_dialog
```
